Settings: PyMailXpress

Welcome to the settings page of PyMailXpress. Here, you can configure and manage your email sending preferences.

Project Overview:

PyMailXpress is a Django web application designed to simplify your email sending experience.

Getting Started:

  1. Ensure that your email account has two-factor authentication enabled for security.
  2. Navigate to Google Account and search for "App Passwords".
  3. Create an app password named "PyMailXpress" or any other name you prefer.
  4. Copy the generated app password.
  5. Clone this repository to your local machine: git clone https://github.com/your-username/pymailxpress.git
  6. Install dependencies: pip install -r requirements.txt
  7. Run the development server: python manage.py runserver

Usage:

  1. Access PyMailXpress in your web browser at http://localhost:8000.
  2. Enter your email address and the generated app password in the provided fields.
  3. Enter a list of recipient email addresses where you want to send the emails.
  4. Enter variables you want to use in your email content.
  5. Write your email content, using the {var} format to insert variables.
  6. Click the "Send" button to send the emails.

Next Steps:
